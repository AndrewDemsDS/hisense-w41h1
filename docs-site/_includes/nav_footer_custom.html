{%- comment -%} The dark-mode toggle. Lives in the sidebar footer so it is reachable from every page without taking space in the nav itself. jtd.setTheme() is the theme's only documented switching API (per its customization docs); there is no native "auto" or prefers-color-scheme support in just-the-docs, which is why this is hand-rolled. The href fallback keeps the button working if that API changes. The light scheme is named "default", not "light", in the generated CSS this site serves. Both files happen to exist, so the pattern below accepts either rather than assuming one. The choice is stored in localStorage and read back by head_custom.html on the next page load, which is what makes the preference stick without a flash. {%- endcomment -%}
